üìã Setup Instructions

Before using the demo:

  1. Install ONNX Runtime: npm install @onnxruntime/web
  2. Rebuild project: npm run build
  3. Download a Piper model from HuggingFace
  4. Place the .onnx file in /models/ folder (create if needed)
  5. Update the model path in the Demo tab
  6. Click "Initialize Synthesizer" button
  7. Enter text and click "Synthesize & Play"

Popular Models:

  • en_US-lessac-medium - Good quality (22MB)
  • en_US-lessac-low - Lower quality (2.2MB)
  • en_US-lessac-high - High quality (80MB)
  • en_GB-alan-medium - British English (22MB)

Troubleshooting:

  • Check browser console for errors (F12)
  • Ensure model file exists and path is correct
  • Clear browser cache if cache issues occur
  • Use developer tools to inspect network requests
